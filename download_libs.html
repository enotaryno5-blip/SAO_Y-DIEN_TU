Cài Đặt Chế Độ Offline

Tải 6 file này và lưu vào cùng thư mục với file index.html.

Lưu ý quan trọng:
- Nếu cột trạng thái báo "Lỗi" (màu đỏ), hãy bấm nút "Tải thủ công" màu xanh hiện ra ngay bên cạnh.
- Khi tải thủ công: Nhấn chuột phải vào trang web hiện ra -> Chọn "Lưu thành..." (Save as...) -> Đặt tên đúng như cột "Tên file".
